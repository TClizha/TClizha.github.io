器件概述

二维电子气与GaN器件结构

2DEG

在AlGaN和GaN的交界处存在着一层二维电子气（2DEG），当施加电压时,这种2DEG可以有效地传递电子，这便是GaN器件导电性的来源

通过在栅源极之间施加负压，可以将2DEG抵消，使得导电通路中断

利用这个原理可以很方便地制成耗尽型器件

耗尽型器件常态为通路，必须施加负压才能维持断态，实际应用起来不方便，于是出现了如下的增强型结构

这种结构使得GaN器件常态下断路，只有在栅源间施加正电压才能导通

常用的增强型GaN器件结构有两种，分别是pGaN Gate Enhancement‐Mode Structure和Hybrid Normally Off Structures

pGaN Gate Enhancement‐Mode Structure

p掺杂的GaN层产生的正电压大于2DEG的挤压电压，使得2DEG被耗尽

Hybrid Normally Off Structures

另外一种做法是将耗尽型GaN管和增强型MOS管串联

由于增加了一个MOS管，这种结构势必会增加系统的损耗，因此这种结构适合用于GaN导通电阻远大于MOS管的情况

由于高导通电阻意味着高耐压，这种方案仅适用于高压系统（高于200V）

反向导通

2DEG流过电流不分正负，使得增强型GaN同样可以反向导通，其原理与正向导通相似

只要栅源极施加的负压大于VthV_{th}Vth​，2DEG就会联通，反向电流流过2DEG

上面介绍的MOS混合结构同样可以流过反向电流，不过反向电流要同时流过MOS的体二极管和耗尽型GaN

器件参数

击穿电压BVDSSBV_{DSS}BVDSS​

用于描述器件承受关断电压的能力

初了这个参数外，芯片厂商的手册里还可能会有高于击穿电压的瞬态电压，这意味着器件可以在短时间内承受更高的关断电压，这一参数和MOSFET的雪崩能力有些相似

泄漏电流IDSSI_{DSS}IDSS​

器件处于关断状态时，仍存在小的泄漏电流IDSSI_{DSS}IDSS​

该参数与温度和VDSV_{DS}VDS​都正相关

导通电阻RDS(on)R_{DS(on)}RDS(on)​

晶体管的导通电阻是指其所有组成部分电阻之和

RDS(on)R_{DS(on)}RDS(on)​的每个组成部分都会随温度变化，因此RDS(on)R_{DS(on)}RDS(on)​的温度系数取决于GaN器件的设计

RDS(on)R_{DS(on)}RDS(on)​还与IDI_DID​和VGSV_{GS}VGS​有关

阈值电压VthV_{th}Vth​

GaN器件的VthV_{th}Vth​基本不随温度变化，然而MOS混合结构的阈值电压取决与MOS的阈值电压，而MOS的VthV_{th}Vth​具有负的温度系数

电容与电荷

与FET相关的电容主要有CGSC_{GS}CGS​、CDSC_{DS}CDS​和CGDC_{GD}CGD​，通过对这些电容充电可以控制FET的开通与关断

手册中可能会给我们输入端电容CISSC_{ISS}CISS​或输出端电容COSSC_{OSS}COSS​，上述电容的换算如下所示

  • COSS=CGD+CDSC_{OSS}=C_{GD}+C_{DS}COSS​=CGD​+CDS​
  • CISS=CGD+CGSC_{ISS}=C_{GD}+C_{GS}CISS​=CGD​+CGS​
  • CRSS=CGDC_{RSS}=C_{GD}CRSS​=CGD​

这些电容不是固定值，而是对应的极电压的函数，因此使用时通常将其积分，得到对应的所需电荷QGQ_GQG​、QGSQ_{GS}QGS​、QGDQ_{GD}QGD​、QOSSQ_{OSS}QOSS​等，这些值不仅和器件有关，还和需要达到的电压有关

反向传导

虽然不像MOS那样有体二极管，但2DEG为GaN器件提供了反向导通的通道

正反电流都流过2DEG，因此正向导通和反向导通的RDSR_{DS}RDS​具有相同的值和温度系数

GaN的驱动

驱动过程

与MOSFET相似，GaN的驱动就是给其寄生电容充电使之达到要求电压

如图，GaN器件的开通过程可以分为四个部分，每部分都有对应的所需电荷

  • VGSV_{GS}VGS​从零达到VthV_{th}Vth​，对应QGS1Q_{GS1}QGS1​
  • VGSV_{GS}VGS​达到平台电压，在这期间IDSI_{DS}IDS​开始增大，对应电荷QGS2Q_{GS2}QGS2​
  • VGSV_{GS}VGS​保持平台电压，在这期间VDSV_{DS}VDS​逐渐减小至零，器件完全导通，对应电荷QGDQ_{GD}QGD​
  • VGSV_{GS}VGS​达到稳态栅极电压

QGQ_GQG​为驱动过程中总的充电电荷，QGSQ_{GS}QGS​为将栅极电压加至平台电压所需电荷

为了描述器件优劣，定义品质因数FOM，其值为QGQ_GQG​和RDS(on)R_{DS(on)}RDS(on)​的乘积

栅极驱动电压

与MOS最高可达±18V\pm18V±18V的栅极电压不同，许多GaN器件栅极最大电压只有+6V/-5V

与此同时，GaN器件要达到完全开通需要+4V以上的电压，一般使用时将稳态栅极电压设定为+5V

为了满足这一要求，通常将栅极驱动器以接近临界阻尼的方式接通回路

器件开通和关断时回路如图所示

开通时，栅极驱动器、晶体管、栅极驱动旁路电容（CVDDC_{VDD}CVDD​）以及其线路电感组成LCR谐振电路

为了严格抑制该电路，防止过冲，线路总的等效电阻必须满足下式

RG(eq)≥4LGCGSR_{G(eq)}\geq\sqrt{\frac{4L_{G}}{C_{GS}}} RG(eq)​≥CGS​4LG​​​

对于下降沿，-5V实际上不存在实际限制，为了更快关断允许一些负振荡

此外，由于少了CVDDC_{VDD}CVDD​，线路此环路的感抗更小

不过要注意负震荡随后的正向震荡不要超过VthV_{th}Vth​

由于开通和关断的要求不同，可以使用两个不同的栅极电阻来独立地调整回路阻尼

自举和浮动电源

驱动半桥时，我们常用下面的结构驱动

当底边导通，高边关断时，电源总线通过二极管给CBootC_{Boot}CBoot​充电；底边关断后，DBootD_{Boot}DBoot​关断，CBootC_{Boot}CBoot​提供电压驱动高边器件

理想情况下，CBootC_{Boot}CBoot​会被充电至总线电压减去二极管导通电压（大约0.7V）的大小

但实际应用中，在下管关断而上管尚未开通时，会有电流反向流过下管进行续流，在下管产生的压降会使得CBootC_{Boot}CBoot​被过充电（VDD−VDBoot+VSDV_{DD}-V_{D_{Boot}}+V_{SD}VDD​−VDBoot​​+VSD​），如果算上线路寄生电感，过充会更大

这一过充可能会影响上管驱动，损坏上管

有三种方法可以减少过充：

  • 在开关互补的应用中，减少死区时间可以减少电容被过充的时间
  • 在下管反并联一个肖特基二极管以减小过充电压
  • 在开关不互补的应用中，使用专用驱动芯片

dv/dtdv/dtdv/dt抗性

低端管漏极可能会有较大的dv/dtdv/dtdv/dt产生，这一dv/dtdv/dtdv/dt会使CDSC_{DS}CDS​充电，连带着CGD和CGSC_{GD}和C_{GS}CGD​和CGS​都会充电，使得VGS>VthV_{GS}>V_{th}VGS​>Vth​，电路误导通

假设CGDC_{GD}CGD​电流全部流过CGSC_{GS}CGS​，则只有密勒比QGD/QGS1>1Q_{GD}/Q_{GS1}>1QGD​/QGS1​>1时，才能保证下管不会误导通，然而实际的GaN管的密勒比很少有小于1的

为了防止误导通，通常在驱动处增加下拉电阻RSinkR_{Sink}RSink​增加电流回路，分流流经QGDQ_{GD}QGD​的电流，防止误导通

此外，在上下开关互补的应用中，可以调整死区时间，使上管恰好在下管关断震荡为负时开通，不过这一方法使得时许固定，不易实现

di/dtdi/dtdi/dt抗性

如图所示，栅极驱动回路和功率回路存在共源电感（CSI）

在开关管关断时，高di/dtdi/dtdi/dt使得共源电感感应出电压，这一电压使得CGSC_{GS}CGS​被驱动到负值，并经阻尼不足的LRC环路谐振得到正的震荡电压使开关管导通

通过增加RSinkR_{Sink}RSink​的阻值来增加环路的谐振可以解决这一问题，但增大RSinkR_{Sink}RSink​会对dv/dtdv/dtdv/dt抗性产生负面影响

另一个方法是改进封装和电路以减小CSI，比如将源极分为两路，一路作为驱动回路，一路作为功率回路

逻辑信号失真

该图展示的是将CSI最小化后的电路，di/dtdi/dtdi/dt在CSI产生了感应电压

当逻辑电路和功率回路共地，驱动电路的地是源极，由于源极和地之间的电压波动，逻辑电路的输出相对驱动电路会产生波动，若波动足够大甚至会影响逻辑输出值

最佳解决方法是将逻辑电路的地接到源极上，但是对于含有多个低端开关管的电路来说并不实用

增加一个RC滤波电路可以有效地滤除这些杂波，但会带来延迟问题和宽度失真

第二种方法是增加电平移位器或隔离器，不过会增加电路元件数量

在半桥电路中，高端驱动会在高电平和低电平之间转换，电容上电压改变会产生共模电流

共模电流同样会影响逻辑输出，严重时改变逻辑状态

减小共模电流，可以通过选用电容较小的器件或和减小回路面积来实现
